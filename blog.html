Question : Discuss the scope of var, let, and const ?

Answer :The scope of a var variable is functional scope. The scope of a let variable is block scope. The scope of a const variable is block scope. It can be updated and re-declared into the scope.

Question : Tell us the use cases of null and undefined ?

Answer :If the user is not logged in, you could assign the value null to the user variable to indicate that there is no logged-in user. On the other hand, undefined is typically used when you want to check if a variable has been assigned a value or not.

Question :What do you mean by REST API ?

Answer :A REST API is an API that conforms to the design principles of the REST, or representational state transfer architectural style. For this reason, REST APIs are sometimes referred to RESTful APIs.
